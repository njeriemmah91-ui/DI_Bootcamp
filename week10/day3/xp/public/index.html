🔐 JWT Auth

Welcome to JWT Authentication Demo

This is a complete implementation of JWT authentication using Node.js and Express.

Features:

  • ✅ User Registration with Validation
  • ✅ User Login with Password Hashing
  • ✅ JWT Token Generation and Verification
  • ✅ Refresh Token Implementation
  • ✅ Secure HTTP-only Cookies
  • ✅ Protected Routes
  • ✅ Token Revocation on Logout

How it works:

  1. Register a new account or login with existing credentials
  2. Your access token is stored securely in an HTTP-only cookie
  3. Access your profile to see authenticated endpoints
  4. Your token automatically refreshes when needed
  5. Logout to revoke your refresh token
